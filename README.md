# Posts app
### Скрипты
- ``npm run dev`` - запускает проект в режиме разаботки;
- ``npm run build`` - собирает проект;
- ``npm run lint`` - запускает eslint и выводит ошибки и их количество;
- ``npm run format:fix`` - запускает prettier и форматирует код;
